Behind the scenes¶

The dataset information is structured in JSON format. The underlying JSON information can be displayed on the browser by adding /export/json to the URL. As an example

  • a portal record of a simulated file

  • the underlying JSON

The JSON files are recorded in the open data portal github area in the records subfolder.

The structure of the JSON information is defined in the schema definition record-v1.0.0.json.
